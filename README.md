# cryptochat
A fun freelance crypto project ! 

Django-backend :  
  1. django rest framework
  2. Celery for automatic scraping
  3. Beautiful Soup for scraping crypto data
  4. React front end
  5. Firebase for building the chat functionality
  6. MaterialUI 
  7. lots of CSS issues :)
